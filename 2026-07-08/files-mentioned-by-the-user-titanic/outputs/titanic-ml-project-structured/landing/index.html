Titanic ML

Un proyecto de aprendizaje supervisado para predecir supervivencia e interpretar los factores que mas influyeron durante el naufragio.

Ver app Ver notebook

Descripcion

Un problema historico convertido en un modelo explicable

El proyecto analiza si variables como sexo, edad, clase del boleto, tarifa, puerto de embarque y tamano familiar permiten estimar la probabilidad de supervivencia de un pasajero del Titanic.

Objetivos

Alcance del proyecto

Explorar los datos

Revisar distribuciones, valores faltantes y patrones de supervivencia.

Preparar variables

Limpiar datos, codificar categorias y crear `FamilySize` e `IsAlone`.

Modelar

Entrenar Regresion Logistica y Regresion Lineal comparativa.

Desplegar

Publicar una app Streamlit y documentar el flujo completo.

Metodologia CRISP-ML

Flujo de trabajo

  1. Comprension del problema
  2. Comprension y exploracion de datos
  3. Preparacion e ingenieria de caracteristicas
  4. Construccion de modelos
  5. Evaluacion e interpretacion
  6. Despliegue con Streamlit y GitHub

Exploracion

Variables clave del analisis

La hipotesis principal es que `Sex`, `Pclass`, `Age`, `Fare` y `FamilySize` explican una parte importante de la supervivencia. El notebook incluye graficos para comparar estos patrones.

Modelos

Resultados esperados

Clasificacion Regresion Logistica

Accuracy, Precision, Recall, F1-score, matriz de confusion y ROC-AUC.

Regresion Regresion Lineal

MAE, MSE, RMSE y R2 para predecir la tarifa del boleto.

Interpretacion Coeficientes

Variables con mayor peso positivo o negativo en la supervivencia.

Aplicacion interactiva

Predice la supervivencia de un pasajero

La app permite ingresar sexo, edad, clase, tarifa, familiares y puerto de embarque para mostrar probabilidad, prediccion y factores influyentes.

Conclusiones

Lectura final

La Regresion Logistica es el modelo adecuado para la variable binaria `Survived`. La Regresion Lineal cumple una funcion academica para comparar una tarea continua con una tarea de clasificacion.
